JavaEE サンプルアプリケーション
---------------------------

## 概要

[@IT連載記事](http://www.atmarkit.co.jp/ait/articles/1507/02/news012.html) で使用するサンプルアプリケーションのJavaEE(JSF)版

## 環境

+ JDK8
+ Glassfish4.1
+ JavaEE7

## 導入方法
このサンプルでは、アプリケーションサーバーの認証機能を使用しているため、
アプリケーションサーバーの設定を行う必要があります。

実行するには、warファイルをビルドして、
アプリケーションサーバーのコマンドでも可能ですが、
NetBeansなど IDEを使用した方が楽です。
